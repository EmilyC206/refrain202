## Cursor Cloud specific instructions

This is a **static HTML/CSS/JS personal blog** generated by [Publii](https://getpublii.com/) and hosted on GitHub Pages at `emilyc206.fyi`.

### Running locally

Serve the repo root with any static file server:

```
python3 -m http.server 8000
```

Then open `http://localhost:8000/`. All pages (homepage, `/defcon-33/`, `/gallery-of-photos/`, `/resume/`, `/tags/`, `/authors/emilyc206/`) will load.

### Caveats

- **Absolute URLs in HTML**: The generated HTML references CSS/JS assets using absolute URLs (`https://emilyc206.fyi/assets/...`). When serving locally, the browser fetches these from the production domain. Styling and scripts still load as long as the production site is reachable.
- **No build step, linter, or test suite**: There is no `package.json`, no build tooling, and no automated tests. The repo contains only the pre-built static output.
- **Content editing** is done via the Publii desktop application, which regenerates the static files into this repo.
